# Tres Monedas - Análisis de Frecuencia

## Descripción

Este programa simula el lanzamiento de 3 monedas y calcula la distribución de frecuencia para el número de caras obtenidas en cada lanzamiento. Se ejecuta un millón de simulaciones para proporcionar una estimación precisa de las probabilidades de obtener 0, 1, 2 o 3 caras. El resultado se muestra en la consola mediante una representación visual con asteriscos.

## Funcionalidades

- **Simulación de lanzamientos**: Realiza un millón de lanzamientos de 3 monedas.
- **Cálculo de frecuencias**: Cuenta cuántas veces se obtuvo 0, 1, 2 o 3 caras en los lanzamientos.
- **Visualización de resultados**: Muestra las frecuencias obtenidas junto con una representación gráfica en asteriscos.
